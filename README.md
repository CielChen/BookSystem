# BookSystem
图书管理系统

--------------------------功能介绍--------------------------
1. 登录： 分为管理员登录以及用户登录. 少量的权限区分
2. 图书列表的显示： 有分页功能，根据用户选择的页数以及每页显示的条数进行筛选
3. 图书列表的排序方式： 根据图书的ID、图书当前剩余的数量进行顺序/逆序排列
4. 图书的管理功能： 管理员拥有图书增、删、改的权限
5. 预约图书： 普通用户和管理员均可以预约图书，并且查看图书详情
6. 查看用户的预约情况
7. 图书检索功能
